Esqueleto

A estrutura básica de um documento HTML (o esqueleto) pode ser gerada no VSCode apenas digitando "!" e dando autocomplete. Ela consiste de um head, onde são feitas as configurações iniciais, e um body, onde efetivamente escrevemos o documento.
HTML não é uma linguagem de programação, é uma linguagem de marcação. Seu nome pode ser entendido como Linguagem de Marcação de Hipertexto, criada na popularização da Internet como principal meio de criar uma rede de páginas interconectadas por Hipertextos (links). A marcação vem da sintaxe da linguagem, onde a estruturação é feita pelo uso de tags.

Hierarquia de cabeçalhos

Os títulos são blocos de texto contidos entre tags do tipo <hn> onde n pode ser um número de 1 a 6.
Cada um desses títulos tem um tamanho de texto na vizualização do HTML, mas isso é só uma questão de hierarquia, nada tem a ver com a formatação. Formatação do texto não se faz com HTML, mas com as CSSs (páginas de estilo). Resumidamente, um título <h3> seria um subtópico de um título <h2> por assim dizer. Vale lembrar que o HTML é quase um rascunho do que será o documento futuramente, ele não apresenta formatação ou estilo algum, apenas conteúdo, imagens, links, textos, o bruto da página.

Algumas ferramentas e tags úteis

Inserindo caracteres especiais

Caso você queira usar caracteres especiais nos títulos, pode fazer isso usando o "&".
Por exemplo, se eu quiser que num título o símbolo "<" seja mostrado apenas como um texto, tal como está sendo agora, e não seja interpretado como uma abertura de tag, eu preciso escrever o e-comercial seguido pelo código do símbolo especial. Nesse caso o código completo é: "&lt;" E fechar com ponto e vírgula.

Quebra de linha

A tag "<br>" poder ser usada para quebrar uma linha dentro de um título. Algumas pessoas fazem essa quebra escrevendo parte do texto num título separado, para emular o efeito, porém, isso não é boa prática.

Comentários

A tag "<!-- -->" pode ser usada para deixar comentários pelo HTML, inclusive espalhadas pelos títulos. Assim como linguagens de programação, aqui comentários não se manifestam no produto final, no caso aqui, não aparecem na página.

Linhas horizontais

Linhas horizontais podem ser inseridas com a tag "<hr>". Algumas dessas linhas já foram usadas nesta página.

Anexando imagens

Com a tag "<img>" podemos linkar imagens à nossa página web. Essa tag requer uma endereço da imagem, ela não precisa estar necessariamente hospedada na mesma página que a página HTML, por isso o endereço pode ser tanto um diretório local, quanto a url de uma imagem hospedade em algum servidor à parte.
