Formatando semanticamente alguns textos

Destaque/ Negrito

Para dar destaque num texto poder ser usada a tag "<strong>", ela é visualimente idêntida ao negrito, que pode ser usado com outra tag, mas que seria apenas formatação de forma, "<strong>" é uma tag novíssima alinhada com os novos objetivos do HTML5 de ser exclusivamente semântico.
Exemplo de texto destacado

Ênfase/ Itálico

Podemos usar a tag "<em>" para enfatizar algo, se forma semântica. Há outra tag também, mas essa aqui é que está alinhada com o caráter semântico do HTML5.
Exemplo de texto enfatizado

Marcação

Podemos destacar um texto com uma cor de fundo usando a tag "<mark>", a cor a gente muda em CSS.
Exemplo de texto marcado

Texto deletado

Para textos que você quer que sejam lidos, mas não considerados, use a tag "<del>".
Exemplo de texto deletado

Texto inserido/ Inserido

Texto sublinhado de forma semântica. Existe outra tag quase em desuso para forma. A tag nova é "<ins>".
Exemplo de texto sublinhado

Texto pequeno

Ainda dá pra achar sentido (semântica) em textos menores que o padrão. Imagine que o HTML é um texto que deve conter em si todo o sentido, o mínimo que precisa ter para ser compreendido, caso fossem desligadas todas as ferramentas de formatação. Num documento assim, às vezes existe sentido em ter letras pequenas, não é apenas uma questão de formatação, mas uma questão de compreensão e função do texto. Para isso existe a tag "<small>".
Exemplo de texto pequeno

Texto sobrescrito e subscrito

Podemos sobrescrever com a tag "<sup>" e subscrever "<sub>".
Exemplo de subscrito: 2H2 + O2 -> 2H2O
Exemplo de sobrescrito: x3 + 3x2 = 5
