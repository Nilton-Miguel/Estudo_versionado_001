Listas Ordenadas

Podemos criar listas ordenadas, ordered lists, com a tag "<ol>", e enumerar itens dessa lista com a tag "<li>", list item, o fechamento desta última tag é opcional no html5.

Listas não Ordenadas

Para listas que são apenas enumerativas, e não ordenadas, unordered list, usamos a tag "<ul>". Que também possui a opção type

Listas de Definição

Listas de definição são usadas para definir termos em glossários, dicionários, etc... Podem ser criadas com a tag "<dl>", definition list. Os itens são criados com 2 pares de tags, as tags "<dt>", define term e "<dd>", define definition.
