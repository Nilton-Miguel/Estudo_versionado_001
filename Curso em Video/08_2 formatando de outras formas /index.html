Formatações Adicionais

Código

Podemos dar sentido de código a algo (sempre pensar HTML5 semanticamente) usando a tag "<code>". Por exemplo:
print("Hello World")

Código pré formatado

A tag "<code>" só dá sentido de código, mas não retém nenhum tipo de formatação, o texto ainda é tratado como qualquer outro texto em html. Se quisermos manter no html a estrutura de qualquer texto, podemos usar a tag de pré formatação: "<pre>". Por exemplo: 

        
    # include <stdio.h>
    int main(){
        printf("Testando formatações em HTML5");
        return 0;
    }

Abreviações

Podemos indicar que algumas palavras são abreviações e deixar um pop up com a versão extendida da palavra usando a tag "<abbr>". Vou usar a palavra Wine de exemplo.

Citações frasais

Podemos dar sentido de uma citação usando a tag "<q>" significando "quote".

Quando não queremos fazer apenas pequenas citações, mas citar blocos de texto e até indicar para o navegador de onde retiramos o trecho, usamos a tag "<blockquote>".
