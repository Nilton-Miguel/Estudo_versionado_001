Formatando basicamente alguns textos

Destaque/ Negrito

Para dar destaque num texto poder ser usada a tag "<strong>", ela é visualimente idêntida ao negrito, que pode ser usado com outra tag, mas que seria apenas formatação de forma, "<strong>" é uma tag novíssima alinhada com os novos objetivos do HTML5 de ser exclusivamente semântico.
Exemplo de texto destacado

Ênfase/ Itálico

Podemos usar a tag "<em>" para enfatizar algo, se forma semântica. Há outra tag também, mas essa aqui é que está alinhada com o caráter semântico do HTML5.
